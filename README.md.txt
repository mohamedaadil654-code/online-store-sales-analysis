# Online Store Sales Analysis Dashboard

## Project Overview
This project analyzes online store sales data using Microsoft Excel.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting

## Dataset
The dataset contains online store order information.

## Dashboard Features
- Total Sales
- Total Profit
- Sales by Category
- Sales by Region
- Monthly Sales Trend
- Interactive Slicers
